# Wavey

[<-- LIVE SITE -->](https://kevinddchen.github.io/wavey/)

Data sources:
 -  [Nearshore Wave Prediction System](https://polar.ncep.noaa.gov/nwps/), downloadable from https://nomads.ncep.noaa.gov/.


## Run locally

```
uv sync
uv run main.py
```

This will create a directory "_site/" which contains the webpage.
